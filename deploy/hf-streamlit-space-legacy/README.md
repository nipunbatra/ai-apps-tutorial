# AI Apps Tutorial

This is a minimal Hugging Face Space README for the built-in Streamlit SDK.

Use it with:

- `app.py`
- `v10.py`
- `requirements.txt`
- `.streamlit/config.toml`

Important:

- This mode is useful for teaching because the config is very small.
- Hugging Face documents the built-in Streamlit SDK as deprecated and recommends Docker for new Streamlit Spaces.
- In the built-in Streamlit Space mode, Streamlit must listen on port `8501`, so do not override the port in `.streamlit/config.toml`.
