# Streamlit Community Cloud

Use the repository root as-is.

## Files to keep

- `app.py`
- `v10.py`
- `requirements.txt`
- `.streamlit/config.toml`

## Local check before deploying

```bash
streamlit run app.py
```

## Deploy steps

1. Push this repository to GitHub.
2. Open Streamlit Community Cloud.
3. Click `Create app`.
4. Choose the repository, branch, and file path `app.py`.
5. Deploy.

## Teaching note

Community Cloud is the easiest first deployment because students only need a GitHub repo plus an entrypoint file.
